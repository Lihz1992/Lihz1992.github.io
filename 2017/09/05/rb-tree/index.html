红黑树

发表于 2017-09-05 |

红黑树

  1. 特殊的二叉查找树，每个节点上都有存储位表示节点的颜色。不论插入还是删除操作都可以在O(lgn)内实现。
  2. 红黑树主要用于存储有序的数据

红黑树特性

  1. 每个节点或者是黑色，或者是红色
  2. 根节点是黑色
  3. 每个叶子节点(NIL)是黑色(空(NIL或null)节点)
  4. 如果一个节点是红色，则它的子节点必须是黑色的
  5. 从一个节点到该节点的子孙节点的所有路径上包含相同数目的黑节点

红黑树基本操作：左旋和右旋

红黑树的基本操作是添加、删除。在对红黑树进行添加或删除之后，都会做旋转操作，以保证红黑树的特性

左旋

对x进行左旋，意味着x将变成一个左节点
