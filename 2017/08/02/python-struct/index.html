Python struct模块

发表于 2017-08-02 |

摘要

This module performs conversions between Python values and C structs represented as Python strings

  1. C语言中，struct结构体用于定义某种结构，包含不同类型数据，方便对某一结构对象进行处理
  2. 网络通信中，传递的数据大多以二进制流(binary data)存在
  3. 当传递非字符串如int、char之类数据，需要某种机制将某些特定的结构体类型打包成二进制流的字符串然后再网络传输，接收端再按某种机制解析出原始结构体数据、
  4. Python struct模块主要用于将Python基本类型值与用Python字符串格式标识的C struct类型间的转化

pack & unpack

struct提供format specifier方式对数据进行打包和解包

格式定义参照 [官方文档] (https://docs.python.org/3/library/struct.html)

字节顺序

struct模块可指定字节顺序，如大端存储、小端存储等特定的字节顺序

pack_into & unpack_from

pack_into、unpack_from方法用于对提前分配好的buffer进行字节的填充，以避免每次pack操作产生新的内存空间

变长字符串的处理
