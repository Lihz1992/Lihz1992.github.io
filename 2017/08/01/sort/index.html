Python 常规排序实现

发表于 2017-08-01 |

Bubble sorted

  1. 冒泡排序: 每个回合，从第一个元素开始和它后面的元素比较，如果该元素比后一元素大就交换;内层循环每次遍历达到的结果是使得剩下的元素中最大的那个放到了序列的“最后”
  2. 时间复杂度$O(n^2)$

选择排序

  1. 选择排序：每个回合，选择最小的元素，置于该次排序起始位置，即第i次排序在待排序列表r[i:]中选择最小记录，与r[i]交换
  2. 时间复杂度$O(n^2)$

插入排序

  1. 每个回合假设当前元素前面的元素有序，把当前位置元素插入到前面的元素序列中的合理位置(因前元素序列有序，可以二分查找所应放置的位置)
  2. 时间复杂度$O(n^2)$

合并排序

  1. 二路合并排序，将原始数据集分成两部分，分别进行排序，然后将排序后的子数据集进行合并
  2. 时间复杂度: $O(nlogn)$
