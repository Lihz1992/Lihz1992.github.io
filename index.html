object reference & gc

发表于 2017-07-24 |

标识、类型和值

每个变量都有标识、类型和值。对象一旦创建，它的标识绝不会变，可以视标识为对象在内存中的地址。is运算符比较两个对象的标识，id()函数返回对象标识的整数表示。

  • CPython中 id()返回对象的内存地址，在其它解释器中可能是别的值
  • ID一定是唯一的数值标注，且在对象的声明周期中绝不会变

逻辑架构

  1. 最上层 连接/线程处理(包括连接处理、授权认证、安全等)
  2. 第二层 查询解析、分析、优化、缓存、所有内置函数(日期、时间、数学和加密函数等)、跨存储引擎的功能(存储过程、触发器、视图等)

closure & decorator

发表于 2017-07-14 |

闭包

Python变量作用域规则

  1. Python 不要求声明变量，但假定在函数定义体中赋值的变量是局部变量；

Base

  1. 闭包指延伸了作用域的函数，包含函数定义体中引用、但不在定义体中定义的非全局变量；

  2. 函数是否为匿名函数不重要，关键是它能否访问定义体之外定义的非全局变量；

  3. 闭包是一种函数，会保留定义函数时存在的自由变量的绑定，当调用函数时，虽然定义作用域不可用了，仍可使用那些绑定的值
