Fluent Python 协程

发表于 2017-08-06 |

协程

yield释义: 产出和让步。yield item产出一个值，提供给next()的调用方，此外，还会做出让步，暂停执行生成器，让调用方继续工作，直至下次调用next()
协程与生成器类似，都是定义体中包含yield关键字的函数。不同在于，协程中yield通常出现在表达式的右边，可以产出值也可以不产出。协程还可能接收调用方通过.send()方法发送的数据
yield关键字还可以不接收或传出数据，不管数据如何流动，yield都是一种流程控制工具，使用它可以实现协作式多任务，协程可以把控制器让渡给中心调度程序，从而激活其它协程
把yield视作控制流程的方式

用作生成器的协程的基本行为

  1. 协程可以身处四个状态中的一个
    • GEN_CREATED 等待开始执行
    • GEN_RUNNING 解释器正在执行
    • GEN_SUSPENDED 在yield表达式处暂停
    • GEN_CLOSED 执行结束
  2. 仅当协程处于暂停状态时才能调用send方法，当协程未激活(处于GEN_CREATED状态时)会抛出TypeError，因而协程的使用，都需要先调用next()函数，这一步称为‘预激协程’
